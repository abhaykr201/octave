Quitting Octave
Shutdown is initiated with the exit or quit commands (they are equivalent). To exit just type exit or quit and window will close like

Getting Help
Using "help" command you can see details of any functionality, in case you need any help. It is use as "help function_name"
help echo

OUTPUT


Comment
'%' is use to comment in Octave Console
% this is a comment
To change Octave name before your code
We use PS1 to remove name of octave before your code
PS1('>> ');

Logical Operations

Equals
gives 0 for false and 1 for true
4 == 3

Not Equals
gives 0 for false and 1 for true
4 ~= 3

And
gives 0 for false and 1 for true
4 && 3

Or
gives 0 for false and 1 for true
1 || 0


Variables

Assignment

if we don't want to print the value at the time of assignment then put a ';' at the end of the code.
a = 5;
to display the value
a
or
disp(a);

To display formated value
a = pi disp(sprintf('%0.3f',a));
